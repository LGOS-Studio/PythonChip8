✨ 核心特性

完整指令集

35 条标准 CHIP‑8 指令全支持，包括绘图、BCD、按键等待、定时器等。

零外部依赖

仅使用 Python 标准库（tkinter / random / time），无需 pip install 任何包。

VMware 风格 UI

左侧虚拟机列表 + 右侧屏幕与寄存器面板，致敬专业虚拟机管理器。

物理键盘映射

CHIP‑8 十六键自然映射至 PC 键盘（1-4/Q-W-E-R/A-S-D-F/Z-X-C-V）。

实时调试视图

PC、SP、I、定时器及 V0~VF 寄存器每帧刷新，便于学习和调试。

ROM 热加载

支持运行 / 暂停 / 重置，随时加载任意 .ch8 文件，即刻游玩经典游戏。
